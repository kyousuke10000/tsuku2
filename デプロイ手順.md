## 📋 デプロイ手順（5分で完了）

### Step 1: Vercelアカウント作成

```
https://vercel.com にアクセス
→ 「Sign Up」でアカウント作成
```

### Step 2: プロジェクトをImport

```
1. Vercel Dashboard で「Add New...」→「Project」
2. 「Import Git Repository」を選択
3. このリポジトリを選択（またはGitHubにpushしてから）
4. 「Deploy」をクリック
```

### Step 3: 完成！

```
自動で以下のURLが生成されます:
https://tsuku2-bot.vercel.app

これで全世界からアクセス可能！
```
